วางลิงก์ Web App จาก Google Apps Script ด้านล่าง
Web App URL
วิธีรับลิงก์ Web App:
1
เปิด Google Sheets ใหม่ → ตั้งชื่อ "สหกรณ์โรงเรียน"
2
คลิกเมนู Extensions → Apps Script
3
วางโค้ดจากไฟล์คู่มือ Code.gs แล้วกด Save
4
กด Deploy → New deployment → เลือก Web app → ตั้ง Execute as: Me, Who has access: Anyone → Deploy
5
คัดลอก Web app URL มาวางช่องบน
